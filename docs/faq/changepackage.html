更换套餐

更换套餐时系统会根据套餐的级别对流量到可用时间进行转化。

升级举例

首先套餐分为低 => 中 => 高三个级别。
如果用户使用的套餐是低级，需要更换为高级套餐，这中间级别差为2级，首先需要换到中级才可以换到高级，每级倍率系数为2.5，2级为5，以5倍的系数对可用时间进行扣除。
则公式为：当前时间 + ((用户剩余时间 - 当前时间) / 5);

降级举例

降级不管从最高级降至最低级，倍率系数均为2.5，以2.5倍的时间进行返还。
则公式为：当前时间 + ((用户剩余时间 - 当前时间) * 2.5);

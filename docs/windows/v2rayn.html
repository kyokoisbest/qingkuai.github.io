V2rayN的使用说明

由于教程编写的时间与当前软件最新版本可能不一致，请注意一些细节上的差异。

下载地址

  • 从Github下载
  • 从CDN下载

安装及使用说明

打开下载地址下载「v2rayN.zip」并解压，随后打开「v2rayN.exe」依次点击「检查更新」然后点击「检查更新v2rayCore」提示下载v2rayCore后确定开始下载。

依次点击「订阅」->「添加」，在「备注」填写方便你记忆的文本，在「地址」填写订阅地址然后点击「确定」，然后点击「订阅」->「更新订阅」即可完成订阅。

双击选择链接节点，然后右键点击「托盘v2rayN图标」选择「启动Http代理」。

完成连接。

答疑

1.可以上谷歌不可以上Youtube是怎么回事？
依次点击「参数设置」->「基础设置」在「自定义DNS」填写「114.114.114.114,8.8.8.8」

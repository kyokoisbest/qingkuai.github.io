V2rayX的使用说明

由于教程编写的时间与当前软件最新版本可能不一致，请注意一些细节上的差异。

下载地址

  • 点击下载

安装说明

打开下载地址下载「V2RayX.app.zip」并解压，随后打开「v2rayX」，第一次提示V2RayX needs to install a small...请点击「Install」，随后再次打开「v2rayX」。

使用说明

打开软件后会在托盘运行，所有的操作都在托盘的图标进行。

依次点击「托盘图标」->「Configure...」。

点击右下角「Advanced...」，选择「Subscription」标签，点击「+」，双击「enter your subscription link here」修改为你的「订阅地址」后按「回车键」保存随后点击「Finish」完成添加。

在「托盘图标」->「Server」进行服务器的选择，随后点击「托盘图标」->「Load core」完成使用。

更新订阅

在「托盘图标」->「Server」->「Update Subscription」进行订阅更新

提示

「托盘图标」->「Global Mode」为全局模式 「托盘图标」->「PAC Mode」为PAC模式 「托盘图标」->「Manual Mode」为Manual模式
